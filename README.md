# floating-youdao-translate
A simple and elegant Youdao Dictionary translation package for Emacs

# Installation

```
(add-to-list 'load-path "/path/to/floating-youdao-translate")
(require 'youdao-translate)
(require 'floating-youdao)
(require 'floating-youdao-interactive)
```
